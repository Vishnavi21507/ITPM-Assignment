# ITPM Assignment - Test Automation UI

## How to Run

1. Install Python (version 3.8 or above)
2. Install dependencies:
   pip install playwright
   playwright install chromium
3. Run the script:
   python image_preview_test.py

## Project Description
This project automates image preview testing using Playwright 
and validates UI behavior on Pixels Suite.
